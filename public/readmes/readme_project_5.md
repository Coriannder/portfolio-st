# Josefina Psicología — Landing Page Premium

Landing page profesional diseñada para servicios de psicología clínica, enfocada en expatriados argentinos en Europa (especialmente Florencia, Italia).

## 🚀 Desafío y Propósito

El objetivo principal fue crear un espacio digital que transmita seguridad, profesionalismo y empatía. La profesional necesitaba una plataforma donde los potenciales pacientes pudieran conocer su enfoque terapéutico y reservar sesiones de forma sencilla, superando la barrera de la distancia física mediante un diseño que inspire confianza.

## 🛠️ Stack Técnico

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router).
- **Lenguaje**: TypeScript para un desarrollo robusto y tipado estricto.
- **Estilos**: [Tailwind CSS 3](https://tailwindcss.com/) con enfoque en diseño responsivo.
- **Componentes**: [Shadcn UI](https://ui.shadcn.com/) para elementos de interfaz accesibles y elegantes.
- **Animaciones**: Framer Motion y Tailwind Animate para transiciones suaves y una experiencia premium.

## 🎨 Decisiones de Diseño

- **Aesthetic**: Minimalista y premium. Se utilizó una paleta de colores suaves y técnica de **Glassmorphism** en tarjetas y navegación para dar una sensación de modernidad y transparencia.
- **Mobile First**: Optimización crítica para dispositivos móviles, incluyendo un sistema de supresión de hovers para evitar interacciones "pegajosas" en pantallas táctiles.
- **Tipografía**: Selección de fuentes que equilibran la seriedad clínica con la calidez humana.

## ⚙️ Características Principales

- **Optimización de Performance**: Uso de Next.js Image para carga eficiente de recursos visuales.
- **Sección FAQ**: Acordeones interactivos para resolver dudas comunes de forma compacta.
- **Integraciones**: Conexión directa vía WhatsApp y estructura preparada para agendamiento automático (Calendly).
- **SEO Estratégico y Técnico**:
    - **Metadatos Dinámicos**: Implementación de la Metadata API de Next.js para títulos y descripciones optimizadas por página.
    - **Open Graph & Twitter Cards**: Configuración de etiquetas para visualización enriquecida al compartir el sitio en redes sociales y WhatsApp.
    - **Estructura Semántica**: Uso estricto de jerarquía de encabezados (H1-H4) y etiquetas HTML5 para mejorar la indexación.
    - **JSON-LD (Schema.org)**: Integración de datos estructurados para "Local Business" y "Person", ayudando a los motores de búsqueda a entender que es un servicio profesional de psicología.
    - **Web Vitals**: Optimización de carga (LCP) y estabilidad visual (CLS) para favorecer el ranking de Google.
    - **Indexing**: Configuración automatizada de `sitemap.xml` y `robots.txt`.
